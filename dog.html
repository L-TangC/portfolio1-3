Concept

'Dognut' was a fundraising product of a dog leash. This video displayed the functions for the product. The main task of this project was to make audiences on social media interested in the product.

Proposal

Client told me to improve to orignal video they filmed in the past. The original vedio's sources were great, but those sources are worth a better edit to increase the quality. So I decided not only to remake the editing, also use motions to create the description titles in the project.

Edit

The first step was to review the files that the client gave me. Then I edited the films based on the script and increased the tempo and qualities for the video. Finally, I made different types of motion graphics for different titles and description to fit each function.
