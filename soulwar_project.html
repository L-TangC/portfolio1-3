Concept

This is a game commercial film. The game is a combination of the card game and Monopoly. The main idea is to highlight the characteristics of the game.

Proposal

"That user wants to play this game" is the main goal of the client. The task of mine is to catch users' eyes. They told me to use lots of effects in the video, and cooperate with soul-stirring music.

Post-production

I used the efficient tool -- After Effect -- to edit this project. Each clip I edited was only 3 seconds, I thought this could make viewers more willing to watch without losing patience.
